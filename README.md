# HTML - Fundamentos

Repositorio de estudos da Formacao HTML Web Developer, parte da minha trilha de desenvolvimento Fullstack. Aqui registro minhas anotacoes, exercicios e pequenos projetos praticos enquanto aprendo HTML.

## Topicos estudados

Estrutura basica de um documento HTML; tags semanticas (header, nav, main, section, article, footer); formularios e inputs; tabelas e listas; links, imagens e midia; acessibilidade (atributos alt e aria); boas praticas e SEO basico.

## Estrutura do repositorio

A pasta exercicios contem exercicios praticos, a pasta projetos contem pequenos projetos e a pasta anotacoes contem resumos de estudo.
